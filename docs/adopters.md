## How to add yourself

Open a PR adding a row to the table below. Include:

- Team / company name
- Use case (one sentence)
- Which adapters you use (`hermes`, `openclaw`, both, other)
- Public link (optional)
- Contact (optional — Discord handle, GitHub username, or email)

We don't verify adoption claims. We trust the community to be honest. Spam entries will be removed.
