# INFINI Color Palette

## Primary

| Color | Hex | Usage |
| --- | --- | --- |
| INFINI Blue | `#06b6d4` | Primary accent, links, active states |
| INFINI Violet | `#7c5cff` | Secondary accent, gradients |
| INFINI Light Blue | `#8494ff` | Gradient start, hover states |

## Backgrounds

| Color | Hex | Usage |
| --- | --- | --- |
| Base | `#0a0e14` | Page background |
| Panel | `#0d1117` | Card background |
| Card | `#11161f` | Elevated card |
| Card Elevated | `#161c28` | Hover state |

## Text

| Color | Hex | Usage |
| --- | --- | --- |
| Primary | `#f8fafc` | Body text |
| Secondary | `#94a3b8` | Muted text |
| Tertiary | `#64748b` | Dim text |

## Status

| Color | Hex | Usage |
| --- | --- | --- |
| Success | `#10b981` | Pass, verified |
| Warning | `#f59e0b` | Retry, caution |
| Danger | `#ef4444` | Fail, error |

## Gradients

```css
/* Primary gradient (logo, buttons) */
linear-gradient(135deg, #8494ff 0%, #06b6d4 100%);

/* Atmospheric haze */
radial-gradient(ellipse 80% 50% at 50% -20%, rgba(6, 182, 212, 0.12), transparent 60%);
```
