# INFINI Typography

## Fonts

| Font | Usage | Weights |
| --- | --- | --- |
| **Plus Jakarta Sans** | Headings, body, UI | 400, 500, 600, 700 |
| **Geist Mono** | Code, traces, terminal output | 400, 500 |

## Scale

| Element | Size | Weight |
| --- | --- | --- |
| H1 (page title) | 3rem (48px) | 700 |
| H2 (section) | 2rem (32px) | 700 |
| H3 (subsection) | 1.5rem (24px) | 600 |
| Body | 1rem (16px) | 400 |
| Small | 0.875rem (14px) | 400 |
| Caption | 0.75rem (12px) | 400 |

## Usage in code

```css
font-family: "Plus Jakarta Sans", -apple-system, BlinkMacSystemFont, sans-serif;
font-family: "Geist Mono", "SF Mono", Menlo, Consolas, monospace;
```

## Usage in Observatory UI

The Observatory uses Plus Jakarta Sans for all UI text and Geist Mono
for trace output, step IDs, and code blocks. This matches the GEM
dashboard aesthetic.
