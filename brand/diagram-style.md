# INFINI Diagram Style

All diagrams in INFINI documentation should follow this style.

## Colors

- **Nodes:** INFINI Blue (`#06b6d4`) with glow
- **Edges:** Muted (`#5b6477`), 1.5px
- **Background:** Panel (`#0d1117`)
- **Text:** Primary (`#f8fafc`) for labels, Secondary (`#94a3b8`) for details

## Node shapes

- **Rectangle** (`rx=8`): steps, agents
- **Circle**: verification gates, budget checks
- **Diamond**: decisions

## Typography in diagrams

- Node labels: Plus Jakarta Sans, 13px, weight 600
- Edge labels: Geist Mono, 11px
- Annotations: Geist Mono, 10px, color `#64748b`

## Example

See `assets/observatory-ui.png` and `assets/spaghetti-to-spec.png` for
the reference aesthetic.

## Tools

- **SVG**: hand-written or generated by scripts
- **Mermaid**: for flowcharts in markdown (use the `dark` theme)
- **React Three Fiber**: for the 3D Observatory graph
