# INFINI Brand

Official design language for INFINI.

## Assets

| File | What |
| --- | --- |
| [`logo.svg`](logo.svg) | INFINI infinity logo with gradient |
| [`wordmark.svg`](wordmark.svg) | "INFINI" text logo |
| [`favicon.svg`](favicon.svg) | 32×32 favicon |
| [`social-banner.svg`](social-banner.svg) | 1280×640 GitHub social preview |
| [`palette.md`](palette.md) | Color palette |
| [`typography.md`](typography.md) | Font choices and scale |
| [`diagram-style.md`](diagram-style.md) | Diagram conventions |

## Usage

- **README banner:** `logo.svg` or `social-banner.svg`
- **Favicon:** `favicon.svg`
- **Social preview:** `social-banner.svg` (set in GitHub repo settings)
- **Documentation:** follow `palette.md` and `typography.md`
- **Diagrams:** follow `diagram-style.md`
