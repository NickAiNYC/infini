# Release Notes — v0.2.0

**Release name:** The Standard Scales

**Release date:** 2026-06-28

## What's new

### Milestone 3: The Standard Scales

INFINI v0.2.0 turns the project from a working local standard into a
credible ecosystem that other frameworks can adopt.

#### Community Adapter SDK

A first-class SDK for external adapter authors. Someone can build an
adapter in under 30 minutes:

- [`sdk/README.md`](../sdk/README.md) — overview and 30-minute path
- [`sdk/adapter-interface.md`](../sdk/adapter-interface.md) — normative contract
- [`sdk/minimal-adapter/`](../sdk/minimal-adapter/) — ~50-line starting point
- [`sdk/testing-guide.md`](../sdk/testing-guide.md) — test before you certify
- [`sdk/certification-guide.md`](../sdk/certification-guide.md) — get the badge
- [`sdk/publishing-guide.md`](../sdk/publishing-guide.md) — share with the ecosystem

#### Community Registry

Git-based adapter registry. No hosted service.

- [`registry/official/`](../registry/official/) — hermes + openclaw
- [`registry/community/`](../registry/community/) — for community adapters
- [`registry/experimental/`](../registry/experimental/) — for adapters in development

#### Marketplace Structure

12 categories with submission templates.

#### Official Design Language

[`brand/`](../brand/) directory with logo, wordmark, favicon, social
banner, palette, typography, and diagram style guides.

#### Documentation Upgrade

Handbook expanded to 17 cross-linked chapters.

#### Benchmark Suite

8 benchmark specs.

#### Contributor Experience

CODEOWNERS, RELEASE_CHECKLIST, issue templates, PR template.
